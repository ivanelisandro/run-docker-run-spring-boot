Description

As you build and run Docker containers, you may find your system cluttered with unused images and containers over time. Managing these resources is essential for keeping your environment clean and efficient. In this stage, you will learn how to stop and remove a running container and delete its associated image. These operations are crucial for effective Docker resource management.

Objectives

  • Stop the container created from the hyper-web-app image.

  • Delete the container associated with the hyper-web-app image.

  • Remove the image with the tag hyper-web-app:latest from the system after its container has been removed.
